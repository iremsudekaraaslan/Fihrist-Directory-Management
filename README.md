# Advanced Directory (Fihrist) Management System & Custom Libraries

This project is a modular directory management system developed in C/C++ that demonstrates advanced memory management, dependency handling, and custom library creation.

## 🚀 Features
* **Modular Codebase:** Core directory algorithms are compiled into a custom **Static Library** (`.lib`) for code reusability.
* **Runtime Optimization:** Transitioned to a **Dynamic Link Library** (`.dll` / `libdynamicfihrist`) architecture for flexible runtime memory allocation.
* **Software Lifecycle (SDLC):** Developed across multiple iterations (`v1` to `v4`) to practice dependency management and version control.

## 🛠️ Tech Stack
* C / C++
* Static & Dynamic Libraries (`.lib`, `.dll`)
* Visual Studio 2022
